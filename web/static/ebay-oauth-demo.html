eBay OAuth Demo

Instructions for eBay RuName OAuth Flow:

  1. Click the "Authenticate with eBay" button below
  2. A popup window will open with eBay's OAuth page
  3. Sign in to your eBay account and grant all requested permissions
  4. Important: After granting permissions, eBay will show a page saying "You can close this window"
  5. A dialog will appear asking you to confirm completion or enter the authorization code
  6. If you see an authorization code on the eBay page, copy and paste it into the dialog
  7. Otherwise, click "Authentication Complete" to proceed
  8. The tokens will be automatically saved and you can test the connection
Note: This app uses eBay's RuName flow, which doesn't automatically redirect back. The system will detect when you've completed authentication and guide you through the final steps.

How to integrate this in your own page:

<!-- Include the OAuth script --> <script src="/static/ebay-oauth.js"></script> <!-- Create a button --> <button id="my-oauth-button">Authenticate with eBay</button> <script> // Setup the OAuth button const button = document.getElementById('my-oauth-button'); setupEbayOAuthButton(button, { onSuccess: (data) => { console.log('Success!', data); alert('Authentication successful!'); }, onError: (error) => { console.error('Error:', error); alert('Authentication failed: ' + error.message); }, onCancel: () => { console.log('User cancelled'); } }); </script>

Or use the manager directly:

const manager = new EbayOAuthManager(); await manager.startOAuthFlow({ onSuccess: (data) => { // Handle success }, onError: (error) => { // Handle error }, onCancel: () => { // Handle cancellation } });
